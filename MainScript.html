Daniil Azarov

Dual Major Student in MS Applied Statistics and PhD Psychology

I'm a dual-major graduate student in Applied Statistics & Psychology at Indiana University, Bloomington, with a strong focus on transitioning into data science. My research background includes machine learning, statistical modeling, and experimental methods applied to human memory, perception, and decision-making.

I work primarily in Python and R, with experience in supervised learning, decision trees, boosting, resampling methods, model evaluation, simulation-based inference, and optimization techniques. I also use JavaScript for behavioral experiments (jsPsych).

I've authored peer-reviewed publications and presented at scientific conferences. I plan to pursue data science roles after completing my degree.

Research Projects

Visual Memorability Research

This project investigates the factors that make visual stimuli memorable, using machine learning approaches to predict and quantify memorability. We employ deep learning models trained on large-scale behavioral datasets to understand what visual features contribute to long-term memory retention. The work combines computational modeling with experimental psychology to bridge the gap between perception and memory systems.

[GitHub] [Related Publications]

Ensemble Perception & Statistical Learning

This research explores how the visual system extracts summary statistics from groups of objects, a process known as ensemble perception. Using psychophysical experiments and computational modeling, we examine how people rapidly perceive average properties of visual scenes. The project also investigates the relationship between ensemble perception and statistical learning mechanisms, with applications to decision-making under uncertainty.

[GitHub] [Related Publications]

Data Science & Industry Projects

Kaggle Competition: Predicting Customer Churn

Developed a machine learning pipeline to predict customer churn for a telecommunications company using gradient boosting and ensemble methods. Implemented feature engineering, hyperparameter tuning with cross-validation, and model interpretation using SHAP values. Achieved top 5% ranking in the competition through careful feature selection and model stacking techniques. Technologies used: Python, scikit-learn, XGBoost, pandas, matplotlib.

[GitHub] [Kaggle Profile]

NBA Player Performance Analytics

Built a comprehensive data analysis pipeline to analyze NBA player performance metrics and predict player efficiency ratings. The project includes web scraping, data cleaning, exploratory data analysis with visualization, and predictive modeling using regularized regression. Created interactive dashboards to visualize player statistics and career trajectories. This project demonstrates end-to-end data science workflow from data collection to model deployment.
